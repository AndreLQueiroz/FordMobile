# 🚀 Ford Racing Simulator Mobile

Uma experiência interativa que combina **simulação de corrida**, **comparação de desempenho automotivo** e **engajamento do usuário através de gamificação**.

---

## 🧠 Visão do Projeto

O **Ford Racing Simulator Mobile** foi desenvolvido com o objetivo de:

* Demonstrar na prática o desempenho de veículos Ford
* Engajar usuários através de uma experiência gamificada
* Comparar dados técnicos com concorrentes de forma visual e intuitiva
* Simular impacto ambiental e eficiência de consumo

> Uma solução que conecta experiência, dados e tomada de decisão em um único app.

---

## 📱 Funcionalidades

### 🏁 Simulação de Corrida

* Registro de tempo de volta
* Validação de formato (mm:ss.SSS ou segundos)
* Feedback instantâneo

### ⚖️ Comparação de Veículos

* Ford vs Concorrência
* Dados técnicos e diferenciais
* Interface clara e visual

### 🌱 Cálculo de Emissão

* Simulação de consumo de combustível
* Comparação entre veículos
* Destaque para eficiência da Ford

### 🏆 Ranking Global

* Ranking por pontos
* Ranking por tempo
* Sistema de gamificação

### 👤 Sistema de Usuário

* Cadastro simples
* Persistência de dados
* Pontuação acumulativa

## ⚙️ Como Rodar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/ford-racing-simulator.git
```

### 2. Acesse a pasta

```bash
cd FordMobile
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npx expo start
```

### 5. Rodar no dispositivo

* 📱 Android: pressione `a`
* 🌐 Web: pressione `w`
